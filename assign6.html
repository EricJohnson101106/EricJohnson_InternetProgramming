Eric Johnson - Assignment 6

Problems

Write a Python program which let the user input a website link and a filename. Then the program will send an HTTP request to the server. Finally, this program will create a new file according to the filename and save the response from the server into that file.

Sample input: www.kennesaw.edu response.txt.

The result is the response will be saved into the file response.txt.
